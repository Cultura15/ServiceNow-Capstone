# Sonya: Pet Shop Needs Management System

**Pet Shop Needs Management** is a role-based application designed to streamline how pet shop orders and inventory are managed. Users can place orders through the Service Portal, clerks can process and manage orders, and managers oversee inventory, approvals, and reports. The system ensures secure access, efficient workflows, and accurate tracking of operations.
<br><br>

*Obectives: To enable Pet Shop clerks and managers to efficiently manage orders and inventory while providing users with a simple Service Portal experience*

## Key Features (Summary)
- **Inventory Management:** Items table viewable by all users; only managers can modify  
- **Order Management:** Users place orders; clerks add orders; managers approve/cancel  
- **Service Portal Modules:** Place Order, My Orders, My Transactions (for users)  
- **Clerk Modules:** Pet Shop Items, All Orders, My Open Orders  
- **Manager Modules:** Approvals, Approved Orders, Reports, Inventory  
- **Notifications:** Stock alerts, item expiration, completed orders  
- **Reports & Data Handling:** Sales and popularity reports, Excel import/export  

---

## Security & Visibility
- Role-based access ensures users see only relevant modules  
- Orders require manager approval for updates/cancellations  
- Users access the system via Service Portal only
